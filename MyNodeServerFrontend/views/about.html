What is Local Node?

Local Node is a locally hosted server that manages and stores files in the local network. It is controlled using a web interface that also provides possibility of discussion on the forum with other people on your network.

Why use Local Node?

You don't have to pay for space using Local Node and there is no internet required, like you would with other similar cloud storages. Local Node is secure, so you do not need to worry about the safety of your files.
